📌 Status

🚧 Currently under development

---

🎯 Objective

This project aims to build a Parking System using Object-Oriented Programming (OOP) concepts in Java.

The goal of this project is to understand and implimented
core oop concepts in a real world scenario.

---

🧠 Concepts Used

- Classes & Objects
- Encapsulation
- Inheritance (planned)
- polymorphism
- Abstraction

---

⚙️ Features (Planned)

-  Parking  a vehicle 
-  Exit the vehicle
-  Display available parking slot
-  Show parking full message
---
